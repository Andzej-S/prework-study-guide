HTML

  • The html element represent the start of html document
  • The meta element connects an external resource to the html document
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS file needs to be linked with html file to take effect
  • * symbol makes it so that ALL of the elements will share the style properties

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: fetches and merges any commits from the tracking remote branch
  • git push: transmit local branch commits to the remote repository branch
